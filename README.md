### Project Description

LocalTix empowers students and communities by making event discovery and participation effortless. It helps organizers increase visibility, improve event management, and boost engagement. By digitizing hyper-local events, the platform fosters stronger community connections and enhances campus and local culture.

### Problem:
Local events like college fests, pop-ups, and community gatherings often struggle with visibility and ticket management. Students and local communities find it difficult to discover nearby events, while organizers lack a simple platform to manage bookings, track attendance, and promote their events effectively. This leads to low participation, poor organization, and missed opportunities for engagement.

### Solution:
LocalTix is a hyper-local event ticketing platform designed to bridge the gap between event organizers and attendees. It enables users to easily discover nearby events, book tickets seamlessly, and manage their attendance, while providing organizers with tools to create events, track bookings, and engage with their audience.

### Key Features
🎟️ Event Discovery: Browse and search for nearby events based on location and interests
📅 Seamless Ticket Booking: Quick and secure ticket purchasing experience
🧾 Organizer Dashboard: Create, manage, and track event performance
🔔 Real-time Updates: Notifications for bookings, reminders, and event changes
💳 Integrated Payments: Smooth payment flow using Razorpay
🤖 Smart Recommendations (Optional AI): Suggest events based on user preferences

### Tech Stack:
    Frontend: Next.js, React
    Backend: Node.js (API routes)
    Database & Auth: Supabase
    Payments: Razorpay
    Deployment: Cloudflare Pages
